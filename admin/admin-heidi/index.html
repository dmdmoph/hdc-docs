    คู่มือการขึ้นทะเบียนผู้ดูแลระบบ (Admin HDC สสจ.)

    แนวทางการดำเนินการจัดทำทะเบียนและทำเนียบผู้ดูแลระบบคลังข้อมูลด้านการแพทย์และสุขภาพ (Admin จังหวัด) ผ่านระบบ Heidi (Line Official)

    🛠️ ขั้นตอนการขึ้นทะเบียนใหม่ (Registration Flow)

    เพื่อให้การขึ้นทะเบียนเสร็จสมบูรณ์ ท่านต้องดำเนินการตามลำดับขั้นตอนดังนี้:

    1. เตรียมเอกสารและลงนาม

    • ดาวน์โหลดแบบฟอร์ม:
      • 📄 แบบฟอร์มแจ้งรายชื่อผู้ดูแลระบบ HDC (ไฟล์ PDF)
      • 📝 แบบฟอร์มแจ้งรายชื่อผู้ดูแลระบบ HDC (ไฟล์ Word)
    • การดำเนินการ: กรอกข้อมูลให้ครบถ้วน และสแกนไฟล์เป็น PDF ที่ลงนามโดยนายแพทย์สาธารณสุขจังหวัดเรียบร้อยแล้ว

    ระดับความลับ

    แบบฟอร์มแจ้งรายชื่อนี้ถือเป็นข้อมูลระดับ [ลับมาก] เมื่อมีการกรอกข้อมูลลงในแบบฟอร์มแล้ว โปรดระมัดระวังในการจัดเก็บและส่งต่อ

    2. เพิ่มเพื่อน Line Official Heidi

    • เพิ่มเพื่อน (Add Friend) ได้ผ่าน 2 ช่องทาง:

      • 📱 คลิกที่ลิงก์: https://lin.ee/Wwj6cex
      • 📸 สแกน QR Code: สแกนภาพ QR Code ของ Line Heidi

    3. อัปโหลดเอกสาร (Upload)

    1. เข้าไปที่หน้าแชท Line Official Heidi
    2. คลิกเมนู "Upload" บน Rich Menu ด้านล่าง
    3. เลือกไฟล์ PDF ที่เตรียมไว้ส่งให้เจ้าหน้าที่ตรวจสอบ และรอผลการอนุมัติผ่านหน้าแชท

    4. ลงทะเบียนข้อมูลส่วนบุคคล (Register)

    เมื่อได้รับข้อความแจ้งผลการอนุมัติผ่านทาง Line เรียบร้อยแล้ว ให้ดำเนินการลงทะเบียนดังนี้:

    1. คลิกเมนู Register: เลือกเมนู "Register" ในหน้าแชท Line Official Heidi เพื่อเข้าสู่หน้าลงทะเบียน

    2. ยืนยันตัวตนด้วย ThaID: ระบบจะนำท่านเข้าสู่กระบวนการพิสูจน์ตัวตนผ่านแอปพลิเคชัน ThaID

    3. กรอกข้อมูลตามระบบ: เมื่อยืนยันตัวตนสำเร็จ ให้ระบุรายละเอียดตามลำดับดังนี้:

      • อีเมล: ระบุอีเมลที่ใช้ติดต่อสื่อสาร (แนะนำ @moph.go.th)
      • เบอร์โทรศัพท์สำนักงาน: ระบุหมายเลขโทรศัพท์ที่ทำงานพร้อมหมายเลขต่อ
      • เบอร์โทรศัพท์มือถือ: กรอกหมายเลขโทรศัพท์ และกดปุ่ม "ขอ OTP" เพื่อรับรหัสยืนยัน
      • OTP: นำรหัสที่ได้รับมายืนยันในช่องที่กำหนด
      • Line: เลือกรูปแบบการระบุตัวตน Line ว่าจะใช้เป็น ID Line หรือ เบอร์โทรศัพท์
      • รูป: คลิกปุ่ม "Choose File" เพื่ออัปโหลดรูปถ่ายหน้าตรง ชุดสุภาพ

    4. ยอมรับเงื่อนไข: ติ๊กเลือกหน้าข้อความ "ยินยอมตามประกาศความเป็นส่วนตัว"

    5. บันทึกข้อมูล: ตรวจสอบความถูกต้องอีกครั้งแล้วกดปุ่ม "ลงทะเบียน"

    5. เริ่มต้นการใช้งานและสื่อสาร

    หลังจากการลงทะเบียนเสร็จสิ้น ช่องทาง Line Official Heidi จะเป็นช่องทางหลักที่ใช้ในการสื่อสาร แจ้งข่าวสาร และประสานงานระหว่างส่วนกลาง (ศทส.) และ Admin จังหวัดอย่างเป็นทางการ

    🚫 การแจ้งยกเลิก (Cancellation)

    ในกรณีที่มีการเปลี่ยนแปลงผู้รับผิดชอบ (ย้าย/ลาออก/เกษียณ) หน่วยงานต้องดำเนินการแจ้งยกเลิกสิทธิ์เดิมทันทีเพื่อความปลอดภัยของข้อมูล

    • ดาวน์โหลดแบบฟอร์มแจ้งยกเลิกรายชื่อ:
      • 📄 แบบฟอร์มแจ้งยกเลิกสิทธิ์ (ไฟล์ PDF)
      • 📝 แบบฟอร์มแจ้งยกเลิกสิทธิ์ (ไฟล์ Word)
    • การลงนาม: ให้ผู้บริหารสำนักงานสาธารณสุขจังหวัดลงนามรับรองในแบบฟอร์ม
    • ช่องทางการส่ง: สแกนเป็นไฟล์ PDF และส่งไปยังอีเมล dmd.moph@gmail.com
    • หัวข้ออีเมล (Subject): แจ้งรายชื่อยกเลิก Admin HDC จังหวัด [ระบุชื่อจังหวัดของท่าน]

    📋 ประกาศการคุ้มครองข้อมูลส่วนบุคคล (Privacy Notice)

    • 📄 ประกาศความเป็นส่วนตัว (Privacy Notice) ฉบับเต็ม
