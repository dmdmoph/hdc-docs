# คู่มือการขอเข้าใช้ระบบ HiPPO

**Health Information Process and Performance Optimization** ระบบประมวลผลและเพิ่มประสิทธิภาพข้อมูลสุขภาพระดับภูมิภาค สำหรับหน่วยงานในสังกัดกระทรวงสาธารณสุข

---

## 👤 คุณสมบัติผู้ขอเข้าใช้งาน
แอดมินของ สสจ. (บน Heidi) ที่ต้องการสิทธิ์ในการ Query ข้อมูลสุขภาพของจังหวัดตนเอง จะต้องมีคุณสมบัติดังนี้:

* **ทักษะด้านฐานข้อมูล:** มีความรู้ความสามารถในการใช้ภาษา HQL หรือ SQL
* **ความรู้ด้านโครงสร้างข้อมูล:** เข้าใจโครงสร้างข้อมูลด้านการแพทย์และสุขภาพ (43 แฟ้ม)
* **การศึกษาเพิ่มเติม:** สามารถศึกษารายละเอียดโครงสร้างข้อมูลได้ที่ [กองยุทธศาสตร์และแผนงาน](https://hdata.moph.go.th/site/)

---

## 🛠️ ขั้นตอนการขออนุมัติสิทธิ์

!!! info "คำแนะนำ"
    การขอใช้งานต้องดำเนินการควบคู่กันทั้งการส่งหนังสือราชการและการลงทะเบียนผ่านระบบออนไลน์

### 1. การดำเนินการด้านเอกสาร
หน่วยงานต้นสังกัดจัดทำหนังสือราชการส่งถึง **ศูนย์เทคโนโลยีสารสนเทศและการสื่อสาร (ศทส.)** พร้อมแนบเอกสาร:

* **บันทึกข้อตกลงการขอเข้าใช้ระบบฯ** (หน่วยงานละ 1 ฉบับ) — *รหัสเอกสาร F-IT-AC-06-01.10*
* **ข้อตกลงและเงื่อนไขการรับสิทธิ์** (1 ฉบับต่อผู้ใช้งาน) — *รหัสเอกสาร F-IT-AC-06-01.11*
* **หมายเหตุ:** สามารถดาวน์โหลดแบบฟอร์มได้จากภายในระบบ

### 2. การลงทะเบียนในระบบ
1. **ลงทะเบียนออนไลน์:** เข้าไปที่ [regis-hippo.moph.go.th](https://regis-hippo.moph.go.th/) และลงทะเบียนด้วย **ThaiD**
2. **รอการตรวจสอบ:** เจ้าหน้าที่ ศทส. จะดำเนินการตรวจสอบเอกสารและอนุมัติสิทธิ์
3. **แจ้งผล:** ระบบจะส่ง **SMS** ยืนยันไปยังโทรศัพท์มือถือเมื่อการอนุมัติเสร็จสมบูรณ์

!!! warning "ข้อกำหนดการใช้ข้อมูล"
    ข้อมูลที่ได้รับสิทธิ์จะเป็นข้อมูลในระดับจังหวัดของหน่วยงานตนเองเท่านั้น

* **ลักษณะข้อมูล:** ข้อมูลไม่ระบุตัวตน (Anonymous Data)
* **ระยะเวลาของข้อมูล:** ข้อมูลย้อนหลังตั้งแต่ปี 2563 จนถึงปัจจุบัน
* **ข้อจำกัดสิทธิ์:** กำหนดให้ใช้งานได้ **สสจ. ละ 3 ท่าน**
* **ช่องทางเข้าใช้งาน:** เข้าใช้งานผ่าน **ThaiD** หรือ **MyMOPH** ได้ที่ [hippo.moph.go.th](https://hippo.moph.go.th/)
* **คู่มือการใช้งาน:** สามารถศึกษาคู่มือฉบับเต็มได้ภายในระบบหลังจาก Login
