# คู่มือการขึ้นทะเบียนผู้ดูแลระบบ (Admin HDC สสจ.)

แนวทางการดำเนินการจัดทำทะเบียนและทำเนียบผู้ดูแลระบบคลังข้อมูลด้านการแพทย์และสุขภาพ (Admin จังหวัด) ผ่านระบบ **Heidi (Line Official)**

---

## 🛠️ ขั้นตอนการดำเนินการ

เพื่อให้การขึ้นทะเบียนเสร็จสมบูรณ์และเป็นไปตามระเบียบรักษาความปลอดภัยข้อมูล ท่านต้องดำเนินการตามขั้นตอนดังนี้:

### 1. เตรียมเอกสารและลงนาม

* **ดาวน์โหลดแบบฟอร์ม**: 
    * 📄 [แบบฟอร์มแจ้งรายชื่อผู้ดูแลระบบ HDC (ไฟล์ PDF)](https://dmd-ict.moph.go.th/api/download/file/6ba805f110aba804f3837e479d2eea414.pdf)
    * 📝 [แบบฟอร์มแจ้งรายชื่อผู้ดูแลระบบ HDC (ไฟล์ Word)](https://dmd-ict.moph.go.th/api/download/file/ff9ae952af2774f6b0829f2e7a98e643.docx)
* **กรอกข้อมูล**: กรอกรายละเอียดในแบบฟอร์มแจ้งรายชื่อผู้ดูแลระบบ HDC ให้ครบถ้วน
* **การรับรอง**: นำแบบฟอร์มให้ผู้บริหารสำนักงานสาธารณสุขจังหวัด (นายแพทย์สาธารณสุขจังหวัด) ลงนามรับรอง
* **เตรียมไฟล์**: สแกนเอกสารที่มีลายเซ็นครบถ้วนเป็นไฟล์ **PDF** เท่านั้น

!!! warning "ระดับความลับ"
    แบบฟอร์มแจ้งรายชื่อนี้ถือเป็นข้อมูลระดับ **[ลับมาก]** เมื่อมีการกรอกข้อมูลลงในแบบฟอร์มแล้ว โปรดระมัดระวังในการจัดเก็บและส่งต่อ

### 3. อัปโหลดเอกสาร (Upload)

1.  เข้าไปที่หน้าแชทของ Line Official Heidi
2.  คลิกที่เมนู **"Upload"** บน Rich Menu ด้านล่าง
3.  คลิกปุ่มเลือกไฟล์เพื่ออัปโหลดไฟล์ **PDF** ที่เตรียมไว้จากขั้นตอนที่ 1
4.  รอเจ้าหน้าที่ตรวจสอบความถูกต้องของเอกสาร (จะมีการแจ้งเตือนผ่านแชท)

---

### 4. ลงทะเบียนข้อมูลส่วนบุคคล (Register)

เมื่อได้รับข้อความแจ้งผลการอนุมัติผ่านทาง Line เรียบร้อยแล้ว ให้ดำเนินการลงทะเบียนดังนี้:

1. **คลิกเมนู Register**: เลือกเมนู **"Register"** ในหน้าแชท Line Official Heidi เพื่อเข้าสู่หน้าลงทะเบียน
2. **ยืนยันตัวตนด้วย ThaID**: ระบบจะนำท่านเข้าสู่กระบวนการพิสูจน์ตัวตน ให้ดำเนินการเข้าสู่ระบบด้วยแอปพลิเคชัน **ThaID** เพื่อยืนยันความถูกต้องของข้อมูลบุคคลก่อนเข้าสู่หน้ากรอกข้อมูล
3. **กรอกข้อมูลตามระบบ**: เมื่อยืนยันตัวตนสำเร็จ ให้ระบุรายละเอียดตามลำดับดังนี้:
    * **อีเมล**: ระบุอีเมลที่ใช้ติดต่อสื่อสาร (แนะนำ @moph.go.th)
    * **เบอร์โทรศัพท์สำนักงาน**: ระบุหมายเลขโทรศัพท์ที่ทำงานพร้อมหมายเลขต่อ
    * **เบอร์โทรศัพท์มือถือ**: กรอกหมายเลขโทรศัพท์มือถือ และกดปุ่ม **"ขอ OTP"** เพื่อรับรหัสยืนยันตัวตนผ่าน SMS
    * **OTP**: นำรหัสที่ได้รับมาคลิกยืนยันในช่องที่กำหนด
    * **Line**: เลือกรูปแบบการระบุตัวตน Line ว่าจะใช้เป็น **ID Line** หรือ **เบอร์โทรศัพท์**
    * **รูปโปรไฟล์**: คลิกปุ่ม **"Choose File"** เพื่ออัปโหลดรูปถ่ายหน้าตรง ชุดสุภาพ (ใช้สำหรับแสดงผลบนทำเนียบหน้าเว็บไซต์)
4. **ยอมรับเงื่อนไข**: ติ๊กเลือกหน้าข้อความ **"ยินยอมตามประกาศความเป็นส่วนตัว"**
5. **บันทึกข้อมูล**: ตรวจสอบความถูกต้องอีกครั้งแล้วกดปุ่ม **"ลงทะเบียน"** เพื่อเสร็จสิ้นขั้นตอน
