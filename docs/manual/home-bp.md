# คู่มือการใช้งานเมนู Home BP (บันทึกข้อมูลการวัดความดันโลหิตที่บ้าน)

คู่มือนี้แนะนำขั้นตอนการขอสิทธิ์และการใช้งานเมนู Home BP บนระบบคลังข้อมูลสุขภาพ (HDC) เพื่อให้หน่วยบริการสามารถบันทึกข้อมูลการวัดความดันโลหิตของผู้ป่วยที่บ้านได้อย่างถูกต้อง

---

## 1. การเข้าสู่ระบบและการเข้าถึงข้อมูลส่วนตัว

เริ่มต้นโดยการเข้าสู่ระบบ HDC ของจังหวัดท่าน และเข้าสู่เมนูข้อมูลส่วนตัวเพื่อเริ่มกระบวนการขอสิทธิ์

1. ไปที่เว็บไซต์ระบบ **HDC** ของจังหวัดท่าน (เช่น `https://hdc.moph.go.th/xxx`) 
2. ทำการ **เข้าสู่ระบบ** ด้วยบัญชีผู้ใช้งาน หรือผ่านระบบ ThaiD / MyMOPH 
3. เมื่อเข้าสู่ระบบแล้ว ให้คลิกที่ **ชื่อผู้ใช้งาน** ที่มุมบนขวา 
4. เลือกเมนู **"ข้อมูลส่วนตัว"** 

---

## 2. ขั้นตอนการขอเพิ่มสิทธิ์การใช้งาน (Home BP)

!!! info "คำแนะนำ"
    สิทธิ์การบันทึกข้อมูล Home BP ไม่ได้ถูกเปิดใช้งานเป็นค่าเริ่มต้น ผู้ใช้ต้องส่งคำขอและรอการอนุมัติจากผู้ดูแลระบบ 

### ขั้นตอนการขอสิทธิ์:
1. ในหน้าข้อมูลส่วนตัว เลื่อนลงมาที่หัวข้อ **"สิทธิการใช้งาน"** 
2. ติ๊กเลือกหน้าหัวข้อ **"บันทึกข้อมูลการวัดความดันโลหิตที่บ้าน (Home BP)"** 
3. คลิกปุ่ม **"+ ขอเพิ่มสิทธิ์"** 
4. เมื่อระบบแสดงข้อความ "ขอเพิ่มสิทธิ์เรียบร้อย กรุณาอัปโหลดเอกสาร" ให้กดปุ่ม **OK** 

### การจัดการเอกสาร:
1. คลิกปุ่ม **"ดาวน์โหลดแบบฟอร์ม"** เพื่อเตรียมเอกสารขอใช้สิทธิ์ 
2. หลังจากกรอกข้อมูลครบถ้วนและสแกนไฟล์แล้ว ให้คลิกปุ่ม **"อัปโหลดแบบฟอร์ม"** เพื่อส่งเอกสารเข้าสู่ระบบ 

!!! warning "สถานะการรอคอย"
    หลังจากอัปโหลดเอกสารแล้ว ระบบจะแสดงข้อความ **"รอแอดมินอนุมัติสิทธิ์"** กรุณารอจนกว่าผู้ดูแลระบบจะทำการตรวจสอบและอนุมัติ 

---

## 3. การใช้งานเมนู Home BP หลังได้รับการอนุมัติ

เมื่อผู้ดูแลระบบอนุมัติสิทธิ์เรียบร้อยแล้ว ท่านสามารถเริ่มบันทึกข้อมูลได้ตามขั้นตอนดังนี้:

1. คลิกที่เมนูหลัก **"นำเข้า/ส่งออกข้อมูล"** 
2. เลือกเมนูย่อย **"บันทึกข้อมูลการวัดความดันโลหิตที่บ้าน"** 
3. ระบบจะแสดงหน้ารายการที่ต้องบันทึกความดันโลหิต 
4. ท่านสามารถใช้ช่อง **ค้นหา** เพื่อระบุ ชื่อ-นามสกุล หรือ เลขบัตรประชาชน ของผู้ป่วยที่ต้องการบันทึกข้อมูล 

!!! success "การส่งออกข้อมูล"
    ในหน้านี้ท่านสามารถส่งออกรายงานข้อมูลการบันทึกในรูปแบบไฟล์ **CSV, Excel หรือ JSON** ได้จากปุ่มเมนูที่อยู่ด้านบนของตาราง
